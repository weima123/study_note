## 一.代理的分类

### 1.静态代理
```
概念：在jvm编译前实现代理类
```

### 2.动态代理
```
概念：在代码运行的过程中，动态生成代理类
```
```
1)   jdk
1.1) jdk代理只能代理有接口的类，根据传入的类实现一个新的类
1.2） 通过实现InvocationHandler接口来实现

2）   cglib
2.1) 没有接口的类的代理只能用cglib，
2.2) cglib通过动态生成类的子类来实现代理，无法代理final修饰的方法
2.3) 通过实现MethodInterceptor接口来实现
```
