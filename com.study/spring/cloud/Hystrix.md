## 二.工作模式
### 1.线程池隔离：
```

```

### 2.信号量隔离：

## 三.hystrix的状态
### 1.closed
```
关闭状态，不使用熔断器
```
### 2.open
```
统计请求的失败占比，当失败占比达到阀值时，打开熔断器,执行fallback逻辑。延迟一段时间（默认5s）后，进入half-open状态
```
### 3.half-open
```
进入half-open状态后，开放部分请求。当请求成功，hystrix进入closed状态；当请求失败，hystrix进入open状态。
```
