#### 1.业务架构
```
  汇率服务对外提供了租户汇率保存、租户汇率查询，以及和币种相关的基础信息（比如币种名称、币种编码、币种符号等）的查询。
```
#### 2.主要功能模块
##### 1.币种相关的基础数据模块：
```
币种编码、币种符号、币种名称
```
##### 2.租户汇率保存：手动录入、汇率自动更新
```
```
##### 3.币种精度模块：
```
处理系统内各种币种换算引起的金额尾差问题
```
##### 4.汇率容差模块：
```
对于员工手动录入的汇率的合理性进行管控;当员工手动录入的汇率和公司维护的汇率差值超过一定的的阀值时，不允许员工录入。
```
